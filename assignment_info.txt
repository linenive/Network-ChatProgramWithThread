2. 조건

    1) 서버(서버 프로그램은 클라이언트가 종료 되어도 종료되면 안된다.)

         - 서버는 "bye"라는 메시지를 받으면 "bye"라고 보낸 클라이언트 소켓만 종료 한다.

         - 서버가 특정 클라이언트로 "bye"라는 메시지를 보내고 보낸 클라이언트 소켓만 종료 한다.

              * 예) server : 1 hi

                      client 1 : hi

                      server : 2 hi

                      client 2 : hi

         - 서버가 "all bye"입력하고 보내면 전체 클라이언트는 "all bye"라는 메시지를 받고 소켓을 종료한다.(접속된 모든 클라이언트 소켓 종료)

         - 서버는 특정 클라이언트를 선택 하여 메시지를 보내야 한다.

         - 전체 클라이언트로 메시지를 보낼 때에는 "all"이라는 문자를 앞에 붙이고 보애야 한다.

             * 예) server : all hi

                     client 1 : hi

                     client 2 : hi

                     client 3 : hi

       2) 클라이언트

           - 클라이언트는 키보드로 부터 "bye"라는 메시지를 보낸 다음 소켓을 종료 한다.
